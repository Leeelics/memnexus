# Experimental: Hierarchical Memory Layers

**Status**: EXPERIMENTAL - Frozen in v1.0

This directory contains experimental implementations of hierarchical memory systems:

- `manager.py` - HierarchicalMemoryManager
- `base.py` - Abstract memory layer classes
- Working/Short-term/Long-term memory layers

These features are based on research papers (MELODI, etc.) but are not actively maintained in v1.0.

For stable memory functionality, use `MemoryStore` from `memnexus.memory.store`.
