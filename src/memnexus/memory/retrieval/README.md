# Experimental: Advanced Retrieval

**Status**: EXPERIMENTAL - Frozen in v1.0

This directory contains experimental retrieval strategies:

- `adaptive.py` - AdaptiveRetriever (SEAKR-inspired)
- `hybrid.py` - HybridRetriever (vector + keyword)

These features are experimental and not actively maintained in v1.0.

For stable retrieval, use basic search from `MemoryStore`.
